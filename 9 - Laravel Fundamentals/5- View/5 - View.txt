üß© STEP 5.2 ‚Äî USING BLADE LAYOUTS (AVOID REPEATED HTML)

Blade Layouts let you reuse one main design (like a header/footer) for all pages.

-------------------------------------------------
STEP 1: CREATE A LAYOUT FILE
-------------------------------------------------
File:
   resources/views/layouts/app.blade.php

Code:
   <!DOCTYPE html>
   <html>
   <head>
       <title>My Laravel Site - @yield('title')</title>
   </head>
   <body>
       <h1>My Laravel Site</h1>
       <hr>

       <div>
           @yield('content')
       </div>
   </body>
   </html>

üß† Explanation:
@yield('title') ‚Üí placeholder for page title
@yield('content') ‚Üí placeholder for page content

-------------------------------------------------
STEP 2: MAKE A PAGE USE THAT LAYOUT
-------------------------------------------------
File:
   resources/views/home.blade.php

Code:
   @extends('layouts.app')

   @section('title', 'Home')

   @section('content')
       <p>Welcome to the Home Page!</p>
   @endsection

-------------------------------------------------
STEP 3: ADD ANOTHER PAGE
-------------------------------------------------
File:
   resources/views/about.blade.php

Code:
   @extends('layouts.app')

   @section('title', 'About')

   @section('content')
       <p>This is the About Page.</p>
   @endsection

-------------------------------------------------

‚úÖ SUMMARY NOTES:
- All HTML pages are in /resources/views/
- Use .blade.php extension
- @extends() ‚Üí use a layout
- @section() ‚Üí define what to show
- @yield() ‚Üí placeholder in layout
- Layouts make all pages share the same header/footer easily
